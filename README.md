# SociaBlock

## Social Media website 

### Sofware to be installed:
* XAMPP download link : https://www.apachefriends.org/download.html
After installation, start Apache and Mysql Services.
---

### Local Setup

Run : 
1. `npm install`
2. `nodemon app.js`
3. webisite starts at : localhost:3000
4. Enjoy!
