### Local Setup

Run : 
1. `npm install`
2. `nodemon app.js`
3. website starts at : localhost:3000
4. Try test emails like "e@e.com","jjjj@pppp.com" etc [no underscores and "." in email ids eg: ee_r@ce.iitr.ac.in]
5. Enjoy!
